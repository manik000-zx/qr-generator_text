# QR Code Generator Web App

A client-side QR Code Generator built using HTML, CSS, JavaScript and GOQR API.

## Features
- Generate QR codes from text
- Live preview
- Download QR image
- Save QR history using LocalStorage
- Profile name & image support

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- GOQR API
- Browser LocalStorage
